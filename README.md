# TaskManager

1. Implemented Technologies
For the development of TaskManager, a robust, scalable, and efficient software architecture was selected, clearly divided between the Frontend (Client Layer) and the Backend (Server Layer), using the following cutting-edge tools:

Frontend (Presentation Layer):

HTML5 & CSS3: Used as the structural and style foundation to ensure a clean, responsive interface that adapts to any screen size (Mobile First).

JavaScript (ES6+): The main client-side engine for dynamic DOM manipulation, real-time event handling, form validation, and asynchronous communication.

Backend & Server (Business Logic Layer):

Node.js: A server-side JavaScript runtime environment that allows for rapid application development thanks to its non-blocking, event-driven I/O model.

Express.js: A minimalist Node.js framework used for routing and creating a robust RESTful API, facilitating the handling of HTTP requests (GET, POST, PUT, DELETE).

Database (Persistence Layer):

MongoDB: A document-based (JSON/BSON) NoSQL database, ideal for its flexibility in storing tasks, subcomponents, users, and dynamic configurations without the rigidity of traditional relational schemas.

Version Control and Deployment:

Git & GitHub: Essential tools for version control, code tracking, and orderly project deployment.
